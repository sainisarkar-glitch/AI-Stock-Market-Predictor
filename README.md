# AI Stock Market Predictor

A GitHub-ready Data Science project using Python, Streamlit, Yahoo Finance data, technical indicators, machine learning, sentiment scoring, and buy/sell/hold recommendation logic.

## Features

- Stock price data download
- Moving averages and RSI indicators
- Machine learning based next-day price prediction
- Buy / Sell / Hold recommendation
- Sentiment analysis section
- Interactive Streamlit dashboard

## Tech Stack

- Python
- Pandas
- NumPy
- Scikit-learn
- XGBoost
- Streamlit
- Plotly
- yfinance

## How to Run

```bash
pip install -r requirements.txt
streamlit run app/streamlit_app.py
```
